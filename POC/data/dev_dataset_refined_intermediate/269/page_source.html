Ausbildungs​management

Wünschen Sie sich manchmal eine Unterstützung bei der Planung und Verwaltung für die Berufsausbildung Ihres Unternehmens? Das Management der Auszubildenden ist ja auch ein regelmässig wiederkehrendes, arbeitsaufwändiges Problem, das sich gut per Rechner unterstützen läßt.



Wir bieten Ihnen dafür unsere Software IAP - Integrierte Ausbildungsplanung an. Je nach dem wie in Ihrem Hause die Ausbildung organisiert ist, finden Sie hier im jedem Fall eine passende Lösung.

Planen Sie primär die Ausbildung von Personen in Abteilungen (wohl der Standardfall), so werden Sie eher auf die Versetzungsplanung für Abteilungen zurückgreifen.

Wenn Sie bei der Ausbildungsplanung Gruppen und nicht einzelne Personen verplanen, dazu verschiedene Räume und/oder Ausbilder berücksichtigen müssen, so benötigen Sie das Add-On Kurs- und Lehrwerkstatt-Planung. Dies trifft vor allem auf Unternehmen mit eigenen Lehrwerkstätten, Gewerbeakademien oder betriebliche Ausbildungszentren zu. [mehr...]









Anwendungs​entwicklung

Wir erstellen in enger Zusammenarbeit mit unseren Auftraggebern maßgeschneiderte Anwendungen: Ob eine einfache Desktop-Lösung oder eine komplexe Datenbankanwendung, wir können Ihnen sicher weiterhelfen.
